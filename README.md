# Satellite-AOD-Retrieval
Python scripts for extracting Aerosol Optical Depth (AOD) at 550nm using MODIS (MCD19A2) or Sentinel-5P (TROPOMI) imagery.
## Features
- Atmospheric data retrieval for urban air quality monitoring.
- Temporal filtering based on satellite overpass windows.
- Spatial clipping to specific area of interest (AOI).

## Requirements
- Google Earth Engine (ee)
- Geopandas
